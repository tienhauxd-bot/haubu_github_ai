# Thread Templates — X.com

## Template 1: Insight Thread (5-8 tweets)

**Khi nào dùng:** Chia sẻ data, research findings, industry trends, lessons learned

**Cấu trúc:**

```
Tweet 1 (Hook): Bold claim với số liệu shock. Tạo gap.
Tweet 2: Bối cảnh — tại sao điều này quan trọng
Tweet 3-6: Supporting evidence — mỗi tweet 1 data point hoặc argument
Tweet 7 (Takeaway): Tóm tắt lesson / framework
Tweet 8 (CTA): Link + hashtags + kêu gọi follow/retweet
```

**Hook patterns:**
- "[Số]% [đối tượng] đang [hành động sai]. Đây là data."
- "Mình đã [hành động] trong [thời gian]. [Kết quả bất ngờ]."
- "[Trend/Tool] đang thay đổi [lĩnh vực]. Nhưng không theo cách bạn nghĩ."

**Ví dụ:**
```
Tweet 1: 90% doanh nghiệp nhỏ fail chuyển đổi số. Không phải vì thiếu tiền.

Đây là lý do thực sự — và cách fix.

Tweet 2: Mình đã tư vấn cho 50+ SMEs trong 2 năm. Pattern lặp lại: Mua tool → dùng 2 tuần → bỏ.

Vòng lặp này tốn trung bình 200M/năm.
```

## Template 2: Step-by-step Thread (5-7 tweets)

**Khi nào dùng:** Hướng dẫn cách làm, tutorial, setup guide

**Cấu trúc:**

```
Tweet 1 (Hook): "Cách [đạt kết quả] trong [thời gian/bước]" + kết quả preview
Tweet 2: Step 1 — giải thích ngắn
Tweet 3: Step 2 — giải thích ngắn
Tweet 4: Step 3 — giải thích ngắn
Tweet 5 (Optional): Step 4-5
Tweet 6 (Result): Kết quả khi hoàn thành
Tweet 7 (CTA): Link tài liệu + hashtags
```

**Hook patterns:**
- "Cách [kết quả] chỉ với [N] bước. Thread:"
- "[Kết quả] mà không cần [điều khó]. Setup trong [thời gian]."
- "Mình vừa [kết quả]. Đây là exact workflow:"

**Ví dụ:**
```
Tweet 1: Cách setup AI automation cho doanh nghiệp nhỏ trong 1 ngày.

Không cần code. Không cần team IT. 4 bước.

Tweet 2: Bước 1: Audit quy trình

Liệt kê mọi task lặp lại hàng ngày. Email, báo giá, follow-up, report.

Nếu bạn làm nó > 3 lần/tuần → automate được.
```

## Template 3: Hot Take Thread (3-5 tweets)

**Khi nào dùng:** Quan điểm mạnh, counter-intuitive insight, myth-busting

**Cấu trúc:**

```
Tweet 1 (Hook): Tuyên bố bold / controversial — tạo reaction
Tweet 2: "Here's why" — evidence đầu tiên
Tweet 3: Evidence tiếp / data point
Tweet 4: Nuanced conclusion — không binary, có depth
Tweet 5 (CTA): Takeaway + link + hashtags
```

**Hook patterns:**
- "[Điều mọi người tin] là sai. Đây là lý do."
- "[Tool/Method phổ biến] đang giết productivity của bạn."
- "Unpopular opinion: [quan điểm ngược]"

**Ví dụ:**
```
Tweet 1: "Học AI" là lãng phí thời gian.

Nghe harsh? Đọc tiếp.

Tweet 2: 90% người "học AI" dành 6 tháng xem tutorials, đọc papers, làm courses.

Output? Zero.

Vì họ học cách dùng tool — nhưng không biết dùng để giải quyết vấn đề gì.
```

## Template 4: Tool/Resource Thread (5-10 tweets)

**Khi nào dùng:** Liệt kê tools, resources, templates, frameworks

**Cấu trúc:**

```
Tweet 1 (Hook): "[N] [tools/resources] giúp [kết quả]. Thread:"
Tweet 2-N-1: 1 tool/resource per tweet — tên + 1 dòng mô tả + use case
Tweet N (CTA): Recap + "Bookmark thread này" + hashtags
```

**Hook patterns:**
- "[N] AI tools mình dùng mỗi ngày để [kết quả]. Free."
- "[N] resources giúp bạn [kết quả] mà 99% người chưa biết."

**Quy tắc per-item tweet:**
- Tên tool/resource: viết HOA hoặc đặt đầu dòng, nổi bật
- 1 dòng mô tả chức năng chính
- 1 dòng use case cụ thể: "Mình dùng để..."

**Ví dụ:**
```
Tweet 1: 7 AI tools mình dùng mỗi ngày. Tiết kiệm 5+ giờ.

Tất cả free hoặc có free tier. Thread:

Tweet 2: CLAUDE CODE

AI coding assistant chạy trong terminal. Không chỉ viết code — nó đọc codebase, chạy tests, fix bugs.

Mình dùng để build full features end-to-end. Tiết kiệm 3-4h/ngày.
```

## Quy tắc chung cho MỌI thread

### Tweet đầu (Hook)
- PHẢI work standalone — người đọc thấy trên timeline phải muốn click
- Có số liệu hoặc kết quả cụ thể
- Kết thúc bằng "Thread:" hoặc xuống dòng tạo curiosity gap
- Max 280 chars — đếm kỹ

### Tweets giữa
- Mỗi tweet có 1 ý rõ ràng — không nhồi nhét
- Đọc riêng vẫn hiểu — không phụ thuộc tweet trước
- Dùng line break giữa các ý trong 1 tweet
- KHÔNG bắt đầu bằng "Tiếp theo..." hay "Ngoài ra..."

### Tweet cuối (CTA)
- Tóm tắt takeaway chính trong 1 câu
- Link (nếu có) — chỉ đặt ở tweet cuối
- Hashtags: max 3, relevant, phổ biến trong Vietnamese tech
- Gợi ý: "Bookmark thread này", "RT nếu hữu ích", "Follow để xem thêm"

### Character count
- Tiếng Việt có dấu tốn nhiều chars hơn — target 200-250 chars/tweet để safe
- Link tự động rút gọn thành 23 chars trên X
- Hashtags tính vào character count

- #AI #AIVietnam #AICoding #AIAutomation
- #ClaudeCode #ChatGPT #Automation
- #OnePersonBusiness #Startup #TechVN
- #ProductivityHacks #NoCode #Framework
