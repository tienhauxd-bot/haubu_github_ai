# Standalone Post Templates — X.com

## Template 1: Data Bite

**Khi nào dùng:** Có 1 số liệu/stat gây shock, insight đơn lẻ powerful

**Cấu trúc:**
```
[Số liệu shock / stat]

[1-2 dòng commentary — góc nhìn riêng]

#Hashtag1 #Hashtag2
```

**Ví dụ:**
```
90% thời gian coding không phải viết code.

Mà là đọc code, debug, và Google.

AI không thay thế dev. AI cắt 90% phần không ai muốn làm.

#AICoding #Automation
```

## Template 2: Quote Card

**Khi nào dùng:** Trích dẫn hay từ video/sách/expert, insight đáng nhớ

**Cấu trúc:**
```
"[Quote / câu nói powerful]"

[Hot take — quan điểm cá nhân về quote này, 1-2 câu]

#Hashtag1 #Hashtag2
```

**Ví dụ:**
```
"Bạn không cần team 10 người. Bạn cần 1 hệ thống AI đúng."

2 năm trước câu này là vision. Hôm nay nó là reality.

Một người + Claude Code + automation = output cả team dev.

#OnePersonBusiness #AI
```

## Template 3: Question Post

**Khi nào dùng:** Muốn tạo engagement, mở discussion, poll ý kiến

**Cấu trúc:**
```
[Câu hỏi provocative]

[Brief context — 1-2 câu tại sao hỏi]

[Optional: quan điểm cá nhân ngắn]
```

**Ví dụ:**
```
ChatGPT hay Claude cho coding?

Mình đã dùng cả hai 6 tháng qua. Kết luận bất ngờ: không phải tool nào tốt hơn — mà là cách bạn prompt quyết định 80% output.

Bạn dùng gì? Reply cho mình biết.
```

## Quy tắc chung cho Standalone Posts

### Format
- Max 280 chars (1 tweet)
- Line breaks tạo nhịp — đừng viết 1 block text dài
- KHÔNG dùng emoji
- Hashtags cuối post, max 3

### Tone
- Punchy, confident, authority
- 1 ý duy nhất — đừng nhồi nhiều ý vào 1 post
- Kết thúc bằng câu đáng nhớ hoặc câu hỏi mở

### Engagement triggers
- Số liệu gây shock → share
- Câu hỏi mở → reply
- Quan điểm bold → quote tweet
- List preview ("Thread:") → click
