# Dạng Listicle — "Top N [Things]"

Video liệt kê N items (tools, repos, tips, resources) với giải thích và demo cho từng cái. Dạng phổ biến nhất cho content AI/tech vì dễ theo dõi, dễ sản xuất, CTR cao.

## Khi Nào Dùng

- Review nhiều tools/repos/resources cùng một chủ đề
- "Top 5 công cụ AI mới nhất tháng này"
- "7 repos GitHub thay đổi cách dùng Claude Code"
- "10 AI tools miễn phí cho solopreneur"

## Cấu Trúc Template

### HOOK (0:00 - 0:15)
Mở bằng bối cảnh + FOMO:
- Nhấn mạnh tốc độ thay đổi ("Mỗi ngày đều có tool mới...")
- Hứa cắt qua noise ("Mình đã lọc giúp bạn...")
- Tease item bất ngờ ("Có vài cái sẽ khiến bạn surprise...")

**Pattern tham khảo (Chase AI):**
> "It feels like every single day we're getting a brand new open-source project on GitHub that changes the way we can use Claude Code. So, to help cut through all of the noise, in today's video, I'm going to show you five of my favorite open-source projects... Some of these are heavy hitters you've probably seen before, but I promise you there's a couple that will surprise you."

### PROMISE (0:15 - 0:25)
Nói rõ:
- Số lượng items
- Tiêu chí chọn (mới nhất, miễn phí, hiệu quả nhất)
- Viewer sẽ được gì sau khi xem

### ITEMS (0:25 - END-2:00)

**Thứ tự sắp xếp items** (chọn 1):
- **Lớn → nhỏ**: Bắt đầu bằng heavy hitter để giữ chân, kết thúc bằng underdog bất ngờ
- **Nhỏ → lớn**: Build anticipation, save best for last — hiệu quả nếu item cuối thật sự WOW
- **Theo chủ đề**: Group items liên quan gần nhau

**Template cho MỖI ITEM:**

```
## ITEM [N]: [TÊN] (X:XX - X:XX)

### Giới thiệu (15-30s)
- Tên tool + context ngắn (bao nhiêu stars, ra khi nào, ai làm)
- ONE SENTENCE mô tả nó làm gì
**VISUAL**: Logo/screenshot tool

### Nó Hoạt Động Thế Nào (30-60s)
- Giải thích cơ chế — đơn giản hóa cho người không technical
- Dùng analogies nếu concept phức tạp
**VISUAL**: Diagram/flowchart hoặc screen recording

### Demo/Proof (30-60s)
- Kết quả thực tế, số liệu cụ thể
- Screenshots, tweets, benchmarks
- "Và kết quả thì... [impressive number]"
**VISUAL**: Screen recording demo hoặc screenshot kết quả

### Ai Nên Dùng / Khi Nào Dùng (15-30s)
- Use cases phù hợp
- Limitations nếu có — tạo trust
- "Nếu bạn đang [situation], thì cái này perfect cho bạn"

### Transition sang item tiếp
- "Okay, giờ item tiếp theo..."
- Hoặc "Nhưng cái tiếp theo mới thật sự là game changer..."
```

### MID-ROLL CTA (~30-40% video)
Đặt SAU item 1 hoặc 2 (item đầu đã deliver value):

```
"Trước khi mình jump vào [item tiếp], mình muốn nhắc nhanh về [product/community].
[1-2 câu value prop liên quan đến video].
Link ở [pin comment/mô tả]. Okay, quay lại..."
```

**Pattern tham khảo:**
> "Before we jump into repo number two, just want to give a quick plug for my Cloud Code Masterclass. This just came out a couple weeks ago and it is the number one place to go from zero to AI dev... And if that's just too much, I also have the free Chase AI community."

### WRAP UP + CTA CUỐI (30-60s)
- Tóm tắt nhanh (1 câu recap)
- Nhắc item yêu thích nhất
- CTA: comment, subscribe, check link

## Ví Dụ Tiêu Đề Hiệu Quả

- "5 Repos Open Source Biến Claude Code Thành Siêu Vũ Khí (Tháng 3/2026)"
- "7 Tool AI Miễn Phí Mà 99% Người Dùng Không Biết"
- "Top 5 Công Cụ AI Mới Nhất — Cái Số 3 Thay Đổi Mọi Thứ"

## Tips

- **Con số lẻ** thường CTR cao hơn chẵn (5, 7, 9)
- **Thêm time constraint** tăng urgency ("tháng này", "tuần qua", "vừa ra mắt")
- **Item bất ngờ** đặt cuối tạo "and one more thing" effect
- **Mỗi item nên độc lập** — viewer có thể skip mà không mất context
- **Thời lượng mỗi item**: 2-4 phút. Video 5 items = 12-18 phút
