# Ultimate YouTube Script Writing Framework

Quy trình viết script YouTube chuyên sâu — từ research đến quality control.

*Dựa trên phương pháp George Blackman và expert insights.*

## Phase 1: Strategic Foundation

### 1. Audience Avatar Development

```
Template:
- Demographics: [tuổi, nghề nghiệp]
- Current State: [đang gặp vấn đề gì]
- Pain Points: [nỗi đau cụ thể]
- Knowledge Level: [trình độ hiểu biết]
- Goals: [mục tiêu mong muốn]
- Language They Use: [từ ngữ họ hay dùng]

Câu hỏi cần trả lời:
- Họ mất ngủ vì điều gì?
- Họ đã thử gì rồi?
- Họ nghĩ mình cần gì vs thực sự cần gì?
```

### 2. Video Purpose Definition

```
Title: [tiêu đề video]

Main Transformation:
  From: [trạng thái trước khi xem]
  To: [trạng thái sau khi xem]

Key Takeaways:
  1. [insight/kỹ năng 1]
  2. [insight/kỹ năng 2]
  3. [insight/kỹ năng 3]

Emotional Journey Map:
  [Cảm xúc đầu] → [Cảm xúc giữa] → [Cảm xúc cuối] → [Hành động]
  Ví dụ: Confusion → Understanding → Confidence → Excitement → Action
```

## Phase 2: Research & Planning

### 1. Content Research

- Nghiên cứu top 10 video performance trong niche
- Phân tích comment sections tìm:
  - Common questions
  - Pain points
  - Objections
  - Success stories
  - Vocabulary người xem dùng

- Tạo swipe file:
  - Hooks hay
  - Transitions mượt
  - Examples hiệu quả
  - CTAs convert cao

### 2. Script Structure Planning

#### Hook Section (30 giây đầu)

Chọn 1 trong 4 loại:

| Loại Hook | Ví dụ |
|-----------|-------|
| **Question** | "Want to know why some creators make 6 figures while others struggle?" |
| **Shocking Statement** | "90% of YouTubers are killing their growth with this one mistake" |
| **Story** | "Three months ago, I was ready to quit YouTube. Then I discovered..." |
| **Preview** | "In the next 10 minutes, I'll show you the exact system that..." |

#### Body Content Framework

Mỗi main point cần 6 phần:

```
1. Open Loop     → "Before I reveal this game-changing strategy..."
2. Why It Matters → "This one technique can double your views because..."
3. What It Is     → "I call this the 'Viral Velocity Formula'..."
4. How To Do It   → "Here are the exact three steps..."
5. Example/Proof  → "Let me show you how [name] used this to..."
6. Transition     → "Now that you understand [X], let's talk about..."
```

## Phase 3: Writing Process

### 1. Hook Writing Formula (30-Second Template)

```
[Pattern Interrupt]
"Want to know the real reason most YouTube channels fail?"

[Credibility Builder]
"After analyzing 1,000+ successful channels..."

[Assumption Break]
"It's not about expensive cameras or perfect editing..."

[Value Promise]
"In this video, I'll show you the simple system that..."
```

### 2. Main Content Development

Mỗi section gồm 3 phần:

#### Setup:
- Present problem hoặc concept
- Share relevant story / example
- Build tension around solution

#### Teaching:
- Giải thích concept đơn giản
- Ví dụ cụ thể
- Actionable steps rõ ràng

#### Proof:
- Show real results
- Share case studies
- Demo implementation

### 3. Engagement Techniques

| Kỹ thuật | Pattern | Ví dụ |
|----------|---------|-------|
| **Tension/Resolution** | "[Belief phổ biến]... nhưng thực tế..." | "Most creators think [X]... but here's what really works..." |
| **Make Viewers Feel Smart** | Cho viewer tự nhận ra | "Take a look at this graph... notice anything unusual? [pause] That's right..." |
| **Story Elements** | Dùng case study cụ thể | "Let me show you how Sarah went from 0 to 100K subscribers..." |

## Phase 4: Production Elements

### 1. Visual Planning

#### B-Roll Checklist:
- [ ] Screen recordings
- [ ] Product demonstrations
- [ ] Real-life examples
- [ ] Supporting graphics

#### Text Overlay:
- [ ] Key statistics
- [ ] Important quotes
- [ ] Step numbers
- [ ] Main points

### 2. CTA Framework

```
1. Reference Current Content
   "Now that you know how to [benefit from current video]..."

2. Create Curiosity Gap
   "But there's one more crucial element you need..."

3. Promise Specific Outcome
   "In my next video, I'll show you how to [specific benefit]..."

4. Clear Direction
   "Click the video on screen now to discover..."
```

## Phase 5: Quality Control

### 1. Script Review Checklist

- [ ] Hook addresses viewer pain point
- [ ] Clear promise established early
- [ ] Smooth transitions between points
- [ ] Specific, actionable examples
- [ ] Simple, conversational language
- [ ] Logical flow maintained
- [ ] All claims supported
- [ ] Clear call to action

### 2. Language Optimization

| ❌ Thay thế | ✅ Bằng |
|------------|--------|
| "Utilize" | "Use" |
| "Implementation" | "Start using" |
| "Methodology" | "Method" |
| "Subsequently" | "Then" |
| "Commence" | "Begin" |

### 3. Final Checks

- [ ] Đọc to — flow tự nhiên?
- [ ] Verify tất cả claims và examples
- [ ] Transitions rõ ràng?
- [ ] Hook deliver đúng title?
- [ ] Time mỗi section
- [ ] Đánh dấu emphasis points
- [ ] Note camera directions
- [ ] Include B-roll notes

## Best Practices Tóm Tắt

### ✅ Luôn include:
- **Context trước concepts** — giải thích bối cảnh trước
- **Examples sau explanations** — ví dụ minh hoạ ngay
- **Proof sau claims** — chứng minh mọi tuyên bố
- **Resolution sau tension** — giải quyết sau khi tạo căng thẳng
- **Clear next steps** — bước tiếp theo rõ ràng

### ❌ Không bao giờ:
- Give away key points quá sớm
- Dùng technical terms không giải thích
- Để câu hỏi không trả lời
- Rush transitions quan trọng

### 🔑 Nhớ:
- Script visual elements kèm theo
- Đánh dấu emphasis points
- Mark pacing changes
- Include emotional beats
- Giữ ngôn ngữ đơn giản
- Test đọc to
- Time mỗi section
