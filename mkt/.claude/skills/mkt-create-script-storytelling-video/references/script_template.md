## ✅ Intro Quality Checklist

### First 5 Seconds:
- [ ] Câu đầu match title?
- [ ] Visual đầu match thumbnail?
- [ ] Credibility thiết lập ngay?

### Hook Structure (20-30s):
- [ ] Có curiosity gap rõ ràng?
- [ ] Có show effort/input bias?
- [ ] Value được promise?
- [ ] Multiple curiosity gaps?

### Technical:
- [ ] Intro ≤ 30 giây?
- [ ] Visual thay đổi mỗi 1.5-2 giây?
- [ ] B-roll hỗ trợ key points?

### Retention Killer Check:
- [ ] KHÔNG có long channel plug?
- [ ] KHÔNG có unnecessary backstory?
- [ ] KHÔNG có slow pacing?
- [ ] Purpose rõ ràng ngay?

### Engagement Test:
- [ ] Xem không sound — hiểu không?
- [ ] Show người khác — hiểu ngay?
