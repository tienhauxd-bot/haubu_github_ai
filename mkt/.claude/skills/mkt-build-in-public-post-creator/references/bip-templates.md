# Build in Public Templates

## Template 1: Progress Update

```
[Hook: Surprising result or number]

Tuần này mình đã [specific action].

Kết quả:
• [Metric 1]
• [Metric 2]
• [Metric 3]

Điều mình học được: [Key lesson]

[1-2 sentences expanding on the lesson]

Nếu bạn cũng đang [related goal], thì đây là tip: [Actionable advice]

[CTA — soft, invite to community]
```

## Template 2: Failure/Lesson

```
[Hook: Admission of failure — bold, honest]

Mình vừa [specific failure].

Chuyện là thế này: [Brief context — 2-3 sentences]

Sai lầm của mình: [What went wrong specifically]

Nhưng mình học được [X] điều:
1. [Lesson 1]
2. [Lesson 2]
3. [Lesson 3]

Bây giờ mình sẽ [what changes going forward].

[Last Dab — powerful closing line]

[CTA — medium, share if relatable]
```

## Template 3: Tool/System Reveal

```
[Hook: "Hệ thống mình dùng để [impressive result]"]

Nhiều bạn hỏi mình dùng gì để [achieve X]. Đây là toàn bộ setup:

[Tool/System 1]: [What it does — 1 sentence]
[Tool/System 2]: [What it does — 1 sentence]
[Tool/System 3]: [What it does — 1 sentence]

Cách mình kết nối chúng: [Brief workflow — 2-3 sentences]

Kết quả: [Specific metric — time saved, output increased]

Quan trọng nhất không phải tool nào — mà là [Key insight about the system thinking].

[CTA — invite to learn more in community]
```

## Template 4: Milestone

```
[Hook: "Từ [starting point] đến [milestone] trong [timeframe]"]

[Brief backstory — where you started — 2-3 sentences]

Con đường không thẳng:
• [Challenge 1 encountered]
• [Challenge 2 encountered]
• [Turning point]

Điều thay đổi mọi thứ: [The key decision or action]

[Numbers/proof — specific metrics]

Bài học lớn nhất: [Insight that others can apply]

[Last Dab — memorable closing]

[CTA — strong, share your milestone too]
```

## Template 5: Behind-the-Numbers

```
[Hook: "Mọi người thấy [visible result]. Đây là thứ đằng sau."]

[The visible result everyone sees — 1 sentence]

Nhưng đây là những gì thực sự xảy ra:
• [Hidden effort 1]
• [Hidden effort 2]
• [Hidden effort 3]

Mình đã [specific struggle or behind-the-scenes detail].

Con số thật: [Real metrics that tell the full story]

Mình chia sẻ điều này vì [reason — to help others have realistic expectations].

[Last Dab]

[CTA — soft, connect in community]
```
