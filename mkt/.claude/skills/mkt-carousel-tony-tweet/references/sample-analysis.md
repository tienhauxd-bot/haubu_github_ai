# Sample Analysis — Drew Huibregtse Carousel

Phân tích 3 slide mẫu user cung cấp (Drew Huibregtse @drewskidigital) để reference khi áp brand Tony Hoang.

## Slide 1 — Prompt Showcase (Prompt 7)

**Cấu trúc:**
- Header: Avatar (đội mũ, stylized) + "Drew Huibregtse" + blue verified + @drewskidigital
- Title: **"Prompt 7 (Scaling Strategy Prompt)"** — bold, ~32pt
- Gray rounded box với prompt content ở trong (gray text, medium weight)
- "How to use it:" label + 1-2 câu giải thích
- "Result you get:" label + 3 bullets với ✓ đen (không phải xanh)
- Carousel arrows 2 bên (indicator — KHÔNG render vào ảnh, IG/X tự thêm)

**Ghi chú cho Tony version:**
- Dùng ✓ xanh `#22C55E` để phân biệt brand
- Title có thể viết VN: "Prompt 7 (Công thức scale lên 10 cuốn sách)"

## Slide 2 — Tweet Card Classic (no title)

**Cấu trúc:**
- Header giống slide 1
- Body: prose dài ~8-10 dòng, có câu bold emphasis ("ONE time.", "20,000+ digital products", "over $100,000")
- Không có box, không có bullets — thuần text storytelling
- Kết bằng 1 câu punchline ngắn ("This is what digital leverage looks like.")

**Ghi chú:**
- Bold emphasis là weapon chính — bold 2-3 cụm key mỗi slide
- Khoảng cách đoạn đều, breathing room

## Slide 3 — Prompt Showcase (Prompt 1)

**Cấu trúc:** Giống slide 1 exactly.

**Điểm chung 3 slides:**

| Element | Style |
|---------|-------|
| Background | Pure white `#FFFFFF` |
| Aspect ratio | ~4:5 portrait |
| Margins | ~80-100px all sides |
| Avatar size | ~80x80px circular |
| Profile name | Bold black, 18-22pt |
| Handle | Gray `#6B7280`, 14-16pt |
| Verified | Twitter-blue filled circle with white check |
| Body text | Near-black, 24-30pt, 1.5 line-height |
| Prompt box | `#F5F5F5` bg, rounded ~12px, padding ~24px |
| Checkmarks | Bold, slightly larger than bullet text |

## Key takeaway cho Tony carousel

1. **Slide 1 luôn là hook** (title bold + sub-context)
2. **Mix 2 layouts** trong 1 carousel: Prompt showcase (actionable) + Tweet card (storytelling) — giữ variety
3. **Bold không thương tiếc** — 2-4 bold per slide
4. **Slide cuối CTA** luôn có "Follow @tranvanhoang.com" hoặc "Join AI Freedom Builders"
5. **Avatar đồng nhất** — reference image lock face qua mọi slide, đó là lý do skill này cần `--avatar` flag mỗi lần chạy
