# Dynamic Results Framework — Score-Based Results Page

The results page changes ENTIRELY based on how the person answered. Three tiers with different tone, insights, and next steps.

---

## Results Page Structure

Every results page has 4 sections:

1. **Big Reveal** — Overall score with visual indicator
2. **3 Insights** — Personalized findings from their answers
3. **Next Steps** — Tier-appropriate CTA (escalates with qualification)
4. **Contact & Social** — Website, social links, support channels

---

## Score Visualization Options

Choose ONE visual for the big reveal:

| Visual | Best For | Description |
|--------|----------|-------------|
| **Speedometer** | Performance/readiness | Gauge showing 0-100% with color zones |
| **Thermometer** | Warmth/readiness | Cold (red) → Warm (yellow) → Hot (green) |
| **Progress Bar** | Completion/maturity | Horizontal bar with % filled |
| **Star Rating** | Simple/consumer | 1-5 stars based on score |
| **Level Badge** | Gamification | Bronze/Silver/Gold badge |

## Tier 1: Low Score (0-40%)

**Tone:** Encouraging, supportive, "lots of room to grow"
**Goal:** Provide free value, build trust, keep them in the funnel

### Big Reveal Template

```
Điểm của bạn: [XX]%

[Visual: Speedometer in red/orange zone]

Bạn đang ở giai đoạn khởi đầu — và đó hoàn toàn không sao!
Phần lớn [target audience] cũng bắt đầu từ đây.
Tin tốt là: bạn có RẤT NHIỀU cơ hội để cải thiện nhanh chóng.
```

### 3 Insights Template

Each insight follows: **Finding → Impact → Quick Win**

```
Insight 1: [Area with lowest score]
Hiện tại [area] của bạn cần được ưu tiên cải thiện đầu tiên.
[Specific finding from their answers]. Điều này đang khiến bạn
[negative impact]. Bước đầu tiên bạn có thể làm ngay:
[one actionable quick win].

Insight 2: [Second area]
[Finding] → [Impact] → [Quick win]

Insight 3: [Positive finding or encouragement]
Điểm tích cực: [something they scored well on or showed awareness of].
Đây là nền tảng tốt để bạn xây dựng tiếp.
```

### Next Steps (Low Score)

**Route to: Free content** — video, guide, article, podcast episode

```
Gợi ý bước tiếp theo dành cho bạn:

Mình đã tạo [free resource name] dành riêng cho người
đang ở giai đoạn như bạn. Trong đó mình chia sẻ:

✅ [Benefit 1 of free resource]
✅ [Benefit 2 of free resource]
✅ [Benefit 3 of free resource]

[Button] Xem ngay miễn phí →

Hoàn toàn miễn phí. Không cần đăng ký thêm.
```

## Dynamic Content Rules

### Content Personalization Matrix

The results page should reference the person's SPECIFIC answers:

| If Q11 (Situation) = ... | Adjust language to... |
|--------------------------|----------------------|
| Individual/employee | Personal productivity framing |
| Freelancer/solopreneur | Business growth framing |
| Manager | Team efficiency framing |
| CEO/Director | Enterprise transformation framing |

| If Q12 (Desired Outcome) = ... | Emphasize in insights... |
|-------------------------------|--------------------------|
| Learning/understanding | Knowledge resources, guides |
| First setup | Step-by-step implementation |
| 70%+ automation | Advanced strategies, tools |
| Full transformation | Strategic consulting, roadmap |

| If Q14 (Preferred Solution) = ... | Route to... |
|---------------------------------|-------------|
| Self-learning | Free content (Tier 1 CTA) |
| Workshop/bootcamp | Group event (Tier 2 CTA) |
| 1-on-1 coaching | Consultation (Tier 3 CTA) |
| Done-for-you | Premium service (Tier 3+ CTA) |

### Override Rules

- If score is High BUT Q14 = "self-learning" → show Tier 2 next steps (not 1-on-1)
- If score is Low BUT Q14 = "done-for-you" → show Tier 3 next steps (they're ready to buy)
- Always respect the person's stated solution preference over score-based routing
