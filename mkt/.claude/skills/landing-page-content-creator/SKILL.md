# Landing Page Content Creator — Online Assessment Lead Generation

Create complete landing page content systems for online assessments that generate high-quality leads. Based on the proven ScoreApp methodology: Landing Page → 15 Quiz Questions → Dynamic Results — achieving 20-40% visitor-to-lead conversion rates.

## System Overview

The online assessment lead generation system has 3 parts:

1. **Landing Page** — Hook + Value Proposition + Credibility + CTA → convinces visitors to start the quiz
2. **Quiz Questions** — Contact info + 10 Best Practices + 5 Qualifying Questions → captures lead data + scores them
3. **Dynamic Results Page** — Big Reveal + 3 Insights + Next Steps → delivers value + routes leads to appropriate offers

## Workflow

### Step 1: Gather Information

Ask for missing info (MUST have all before writing):

1. **Niche/product** — What business, product, or service is this for?
2. **Target audience** — Who takes this assessment? (job title, pain points, goals)
3. **Core frustration** — What recurring frustration does the audience face?
4. **Dream outcome** — What result do they want in the next 90 days?
5. **3 key areas** — What 3 measurable areas will the assessment evaluate?
6. **Credibility** — Who created this? Background, stats, research to reference?
7. **Solutions offered** — What products/services do you offer at different price points?
8. **Language** — Vietnamese (default) or English?

### Step 2: Write Landing Page Content

Follow the 4-section landing page structure. See [references/landing-page-framework.md](references/landing-page-framework.md) for detailed templates.

**Section 1 — Hook** (top of page, first thing visitors see)
Choose ONE hook type:

| Hook Type | Formula | Best For |
|-----------|---------|----------|
| **Frustration Hook** | "Bạn có đang frustration... dù đã làm X?" | Pain-aware audience |
| **Results Hook** | "Bạn đã sẵn sàng để desired result?" | Aspiration-driven audience |

**Section 2 — Value Proposition**
- "Trả lời 15 câu hỏi để đo lường và cải thiện 3 lĩnh vực quan trọng:"
- List the 3 key areas with brief descriptions

**Section 3 — Credibility**
- Who created this assessment and why
- Background, experience, relevant stats
- Research or data that validates the assessment

**Section 4 — Call to Action**
Must include ALL 4 elements:
- Start the quiz (primary action)
- Only takes 3 minutes (time investment)
- Completely free (no risk)
- Get immediate recommendations (instant value)

### Step 3: Write Quiz Questions

Follow the 15-question framework. See [references/quiz-questions-framework.md](references/quiz-questions-framework.md) for templates.

**Part A — Contact Info** (auto-captured)
- Name (required)
- Email (required)
- Location (auto via IP)
- Phone (optional)

**Part B — 10 Best Practices Questions** (scoring questions)
- Yes/No or scale questions about whether they follow best practices
- These generate the score (0-100%)
- Must cover the 3 key areas from the value proposition

**Part C — 5 Qualifying Questions** (sales intelligence)

| # | Question Type | Purpose | Format |
|---|--------------|---------|--------|
| Q11 | Current Situation | Segment the lead | Multiple choice (4-5 options) |
| Q12 | Desired Outcome (90 days) | Understand motivation | Multiple choice (3-4 options) |
| Q13 | Biggest Obstacle | Identify pain point | Multiple choice (3-4 options) |
| Q14 | Preferred Solution | Gauge budget/commitment | Multiple choice (3-4 options) |
| Q15 | Anything Else | Open intel | Free text box |

### Step 4: Write Dynamic Results Page

Follow the results framework. See [references/dynamic-results-framework.md](references/dynamic-results-framework.md) for templates.

Create 3 result tiers based on score:

| Score | Tier | Tone | Next Step |
|-------|------|------|-----------|
| 0-40% | Low | Encouraging, "room to improve" | Free content (video, guide, article) |
| 41-70% | Medium | Positive, "strong foundations" | Group event, webinar, workshop |
| 71-100% | High | Congratulatory, "ready to scale" | 1-on-1 consultation, premium offer |

Each tier includes:
- **Big Reveal** — Overall score with visual (speedometer/thermometer)
- **3 Insights** — Personalized findings based on answers
- **Next Steps** — Tier-appropriate CTA
- **Contact Info** — Social links, website, support channels

### Step 5: Output Format

Deliver as 3 clearly separated sections:

```
## PHẦN 1: LANDING PAGE
### Hook
### Value Proposition
### Credibility
### Call to Action

## PHẦN 2: CÂU HỎI QUIZ (15 câu)
### Contact Info Fields
### Best Practices (Q1-Q10)
### Qualifying Questions (Q11-Q15)

## PHẦN 3: TRANG KẾT QUẢ ĐỘNG
### Tier 1: Score 0-40% (Low)
### Tier 2: Score 41-70% (Medium)
### Tier 3: Score 71-100% (High)
```

## Mandatory Rules

- **Gather ALL info** before writing — never assume niche details
- Landing page hook must be ONE of the two types (Frustration or Results) — not both
- Value proposition MUST list exactly 3 measurable areas
- CTA MUST include all 4 elements (start, time, free, instant value)
- 10 best practices questions MUST cover all 3 key areas (roughly 3-3-4 split)
- 5 qualifying questions MUST follow the exact sequence (situation → outcome → obstacle → solution → open)
- Results page MUST have 3 distinct tiers with different next steps
- Each tier's next steps must match the lead's qualification level
- Q14 (preferred solution) options must imply different budget levels
- If Vietnamese: use brand voice from BRANDVOICE.MD — "mình", "bạn", conversational expert tone
- If Vietnamese: keep power words in English (System, Automation, AI, Framework, Workflow, Template)
- **NEVER write generic copy** — every line must be niche-specific with concrete examples
