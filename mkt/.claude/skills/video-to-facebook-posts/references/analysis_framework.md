# Analysis Framework - Phân tích nội dung video

## Bước 1: Nhận diện loại nội dung

### Loại 1: Liệt kê kiến thức (Knowledge Listing)

**Dấu hiệu nhận diện:**
- Video liệt kê nhiều items: tips, tools, features, steps
- Cấu trúc dạng "X cách...", "X tools...", "Hướng dẫn step-by-step"
- Mỗi mục được giải thích riêng biệt, không so sánh với nhau
- Người xem học được "danh sách những gì cần làm/biết"

**Từ khóa trigger:**
- "hướng dẫn", "cách setup", "step-by-step", "tips", "hacks"
- "features", "tools", "cách dùng", "workflow"
- Các số thứ tự: "thứ nhất", "tiếp theo", "cuối cùng"

**Ví dụ:**
- "5 AI Tools giúp tiết kiệm 10 giờ/tuần"
- "Hướng dẫn setup Claude Code từ A-Z"
- "10 prompt templates cho doanh nghiệp"

---

### Loại 2: Phân tích so sánh (Comparative Analysis)

**Dấu hiệu nhận diện:**
- Video so sánh 2+ thứ: tools, approaches, methods, versions
- Có bảng ưu nhược điểm hoặc phân tích trade-offs
- Kết luận bằng recommendation: "nên dùng X khi..., dùng Y khi..."
- Người xem học được "chọn cái nào và tại sao"

**Từ khóa trigger:**
- "so sánh", "vs", "khác nhau", "ưu nhược điểm"
- "pros and cons", "nên chọn", "khi nào dùng"
- "better", "worse", "trade-off", "difference"

**Ví dụ:**
- "ChatGPT vs Claude - Dùng cái nào?"
- "Agent Teams vs Sub Agents - Khác nhau thế nào?"
- "React vs Vue cho dự án startup"

---

### Loại 3: Hybrid (Kết hợp)

**Dấu hiệu nhận diện:**
- Video vừa liệt kê nhiều items VÀ so sánh chúng với nhau
- Thường là video giới thiệu nhiều options rồi phân tích mỗi cái
- Có cả danh sách lẫn bảng so sánh

**Ví dụ:**
- "3 chế độ agent trong Claude Code - setup và khác nhau thế nào?"
- "5 AI coding tools - phân tích từng cái và nên chọn gì"

---

## Bước 2: Trích xuất Key Insights

### Checklist trích xuất (áp dụng cho MỌI transcript):

| Yếu tố | Câu hỏi | Bắt buộc? |
|---------|---------|-----------|
| **Main Points** | Các điểm chính video muốn truyền tải là gì? | Bắt buộc |
| **Số liệu** | Có con số, %, thống kê cụ thể nào? | Nếu có |
| **Case Study** | Có ví dụ thực tế, demo, kết quả cụ thể? | Nếu có |
| **Pain Points** | Vấn đề/đau đớn nào được nêu ra? | Bắt buộc |
| **Solutions** | Giải pháp/cách giải quyết được đề xuất? | Bắt buộc |
| **Quotes** | Câu nói hay, đáng nhớ từ transcript? | Nếu có |
| **Recommendation** | Kết luận/khuyến nghị cuối cùng? | Bắt buộc |
| **Unique Angle** | Góc nhìn độc đáo so với nội dung phổ biến? | Bắt buộc |

## Bước 3: Xác định góc viết Facebook

### Nguyên tắc chọn góc:
- Mỗi bài viết khai thác **một khía cạnh khác nhau** của video
- Không viết 3 bài nói cùng một ý
- Ưu tiên góc nhìn tạo "gap" - thứ người đọc chưa biết hoặc hiểu sai

### Ma trận góc viết theo loại nội dung:

**Liệt kê kiến thức → 3 góc:**
1. **Actionable**: Chọn 1 tip/tool hay nhất → viết sâu, hướng dẫn áp dụng ngay
2. **Listicle**: Tóm tắt toàn bộ danh sách → format dễ scan trên Facebook
3. **Myth-busting**: Lấy 1 quan niệm sai phổ biến từ video → phản bác

**Phân tích so sánh → 3 góc:**
1. **Comparison**: So sánh trực diện A vs B → bảng ưu nhược rõ ràng
2. **Deep-dive**: Đi sâu vào 1 option → phân tích tại sao phù hợp nhất
3. **Decision framework**: Hướng dẫn cách ra quyết định → "Chọn A khi..., chọn B khi..."

**Hybrid → 3 góc:**
1. **Comparison**: So sánh các items được liệt kê
2. **Actionable**: Chọn item hay nhất → hướng dẫn setup/áp dụng
3. **Overview + Opinion**: Tổng quan + quan điểm cá nhân "Mình chọn X vì..."
