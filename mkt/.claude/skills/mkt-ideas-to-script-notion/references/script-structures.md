# Short-Video Script Structures (v1)

Adapted from `mkt-create-script-short-video` with two additions specific to this skill: **inline `[REF: url]` markers** and a **Last Dab + CTA** suited to the AI Freedom Builders community.

Write **Last Dab first → Hook → middle**.

---

## 1. Before-After

**Use for:** tips, hacks, tool reviews, transformations, before-after result videos.

```
[HOOK — 3 giây đầu]
"Trước đây <vấn đề + con số cụ thể>… giờ thì <kết quả + con số>"
[REF: <asset thể hiện trạng thái cũ hoặc mới>]

[TRƯỚC — Vấn đề]
- Nêu vấn đề người xem đang gặp (cụ thể, đo được)
- Ảnh hưởng tiêu cực

[CẦU NỐI — Giải pháp]
- Bước 1: ...
[REF: <screenshot / clip demo bước 1>]
- Bước 2: ...
- Bước 3: ...

[SAU — Kết quả]
- Kết quả đo lường được
- Cảm xúc / lợi ích

[LAST DAB + CTA]
"<punchline>. Muốn học cách xây đội bot làm 24/7, vào AI Freedom Builders."
```

## 2. Three Acts (Ba Hồi)

**Use for:** personal story, journey, conflict-driven experience.

```
[HỒI 1 — MỞ ĐẦU]
- Bối cảnh ban đầu
- Mình đang làm gì?
[REF: <asset bối cảnh>]

[HỒI 2 — XUNG ĐỘT]
- Vấn đề xuất hiện
- Nỗ lực thất bại
- Căng thẳng leo thang
[REF: <asset thể hiện vấn đề>]

[HỒI 3 — GIẢI QUYẾT]
- Phát hiện / giải pháp
- Cách vượt qua
- Bài học
[REF: <asset kết quả>]

[LAST DAB + CTA]
```

---

## 3. Action

**Use for:** challenges, reactions, tense/unexpected situations, live demos.

```
[MỞ ĐẦU BẰNG HÀNH ĐỘNG]
- Vào thẳng tình huống, không giới thiệu
[REF: <clip mở màn>]

[TĂNG CƯỜNG HÀNH ĐỘNG]
- Thử 1 → chưa được
- Thử 2 → vẫn chưa
- Căng thẳng leo thang
[REF: <asset minh họa thử nghiệm>]

[HÀNH ĐỘNG GIẢM DẦN]
- Giải quyết
- Lời khuyên / bài học

[LAST DAB + CTA]
```

## Picking a Structure

| Idea looks like… | Pick |
|------------------|------|
| "Trước X giờ Y", tool review, productivity hack | **Before-After** |
| "Mình từng…", journey, mistake story | **Three Acts** |
| "Thử xem chuyện gì xảy ra khi…", demo, reaction | **Action** |

When in doubt, default to **Before-After** — it is the most versatile for AI/tech content.

## Hard Brand Rules (enforced before pushing to Notion)

- First person = **`mình`** (never `tui`, never `tôi`).
- Audience = **`bạn`** / **`các bạn`**.
- Specific numbers over vague words ("200 view", not "view thấp").
- No English jargon — only brand names (ChatGPT, Claude, AI, Anthropic, OpenAI, Cursor…) are allowed.
- One `[REF: url]` per reference link the user supplied, placed where that asset best supports the narration. If a reference has no natural beat, attach it to the closest related paragraph rather than dropping it.
- End with a CTA that ties back to the **AI Freedom Builders** community.
