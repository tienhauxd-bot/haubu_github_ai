# Instagram Rules — Carousel

## Caption Rules

| Setting | Value |
|---------|-------|
| Length | 150-300 ký tự (before hashtags) |
| Tone | Professional-clean, knowledge-sharing |
| CTA | "Save lại", "Share cho bạn bè", "Follow @tranvanhoang.com" |
| Hashtags | 20-30 tags, sau caption, cách 5 line breaks |

## Caption Structure

```
[Hook 1 dòng — tóm tắt giá trị carousel]

[1-2 dòng context / promise]

[CTA — Save / Follow / Share]

.
.
.
.
.

#hashtag1 #hashtag2 ... #hashtag30
```

## Instagram Hashtag Strategy

### Tier 1 — High volume (5-8 tags)
Broad reach, competitive:
- #AI #ArtificialIntelligence #Automation #TechTips
- #DigitalTransformation #Productivity #BusinessTips
- #AItools #TechVietnam

### Tier 2 — Medium volume (8-12 tags)
Niche but active:
- #AIautomation #AICoding #NoCode #WorkflowAutomation
- #SmallBusiness #Entrepreneur #StartupLife
- #OnlineBusinessTips #DigitalMarketing #ContentCreator

### Tier 3 — Low volume / branded (5-10 tags)
Very targeted:
- #AIVietnam #OnePersonBusiness #AIForBusiness
- #ClaudeAI #ClaudeCode #AIAgent
- #tranvanhoang #AIFreedomBuilders
- #AITips #AutomationTips #ProductivityHacks

### Mix rule
- Mỗi post: ~5 Tier 1 + ~10 Tier 2 + ~10 Tier 3 = ~25 total
- Rotate tags giữa các posts — không dùng exact same set

## Slide Design — Instagram Specifics

| Element | Value |
|---------|-------|
| Aspect ratio | 1:1 (1080x1080px) |
| Font size (headline) | 48-64px equivalent |
| Font size (body) | 28-36px equivalent |
| Padding | 80-100px mỗi bên |
| Watermark | @tranvanhoang.com ở footer, centered |
| Max text per slide | 40 words |
| Readability | Phải đọc được khi xem thumbnail size |

## Instagram Algorithm Notes
- Carousel posts có reach cao hơn single image ~1.4x
- Saves là signal mạnh nhất — content "save-worthy" perform tốt
- First slide = hook, quyết định engagement rate
- 7-10 slides optimal cho IG carousel
