# TikTok Rules — Carousel

## Caption Rules

| Setting | Value |
|---------|-------|
| Length | 100-200 ký tự |
| Tone | Casual, trending, gần gũi |
| CTA | "Save ngay", "Follow để xem thêm", "Share cho bạn" |
| Hashtags | 3-5 tags, cuối caption |

## Caption Structure

```
[Hook ngắn — 1 dòng punchy]

[CTA ngắn]

#hashtag1 #hashtag2 #hashtag3 #hashtag4
```

## TikTok Hashtag Strategy

### Trending tags (1-2)
Thay đổi theo trend, check TikTok Discover:
- #LearnOnTikTok #TikTokAcademy #fyp #viral

### Niche tags (2-3)
- #AITips #AutomationHacks #TechTikTok
- #OnePersonBusiness #AIForBusiness
- #ProductivityTips #WorkSmarter

### Mix rule
- 1-2 trending + 2-3 niche = 3-5 total
- KHÔNG spam hashtags — TikTok algorithm penalize

## Slide Design — TikTok Specifics

| Element | Value |
|---------|-------|
| Aspect ratio | 9:16 (1080x1920px) |
| Font size (headline) | 64-80px equivalent |
| Font size (body) | 36-48px equivalent |
| Padding | 60-80px sides, 120px top/bottom (avoid UI overlap) |
| Watermark | @tranvanhoang.com ở bottom, above TikTok UI |
| Max text per slide | 30 words (bigger font = less text) |
| Safe zone | Avoid bottom 200px (TikTok UI covers) |

## Tone Differences vs Instagram

| Element | Instagram | TikTok |
|---------|-----------|--------|
| Language | Professional-clean | Casual, trending |
| Slang | Minimal | OK: "chill", "real talk", "slay" |
| Energy | 7/10 | 8/10 — bold, attention-grab |
| Humor | Subtle | More welcome |
| Trend ref | No | Có thể reference trends |

## TikTok Algorithm Notes
- Carousel photos đang được TikTok push mạnh (2024-2026)
- "Save" và "Share" là signals quan trọng nhất
- Educational content perform rất tốt dạng carousel
- 5-8 slides optimal cho TikTok
- Font lớn hơn IG vì xem trên phone fullscreen
