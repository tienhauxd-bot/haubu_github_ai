## Post 3: The Breakdown + Framework Angle

Hệ thống tìm việc kiểu automation - nó build trên **3 framework cốt lõi**:

**1. SCRAPING → PARSING → RANKING**
Hệ thống tìm 700+ listings, parse thông tin job, rank những cái match với profile bạn. Điểm số cao nhất được ưu tiên.

**2. PROFILE MATCHING**
AI analyze resume/skills của bạn, so sánh với requirement của từng job. Match score thấp thì skip. Match score cao thì proceed.

**3. PERSONALIZATION AT SCALE**
Đây là cái clever nhất. Thay vì viết 1 generic cover letter rồi copy-paste 100 lần, hệ thống tạo resume + cover letter unique cho mỗi job. Why? Vì nó có access vào: job description, company info, role requirements → nó viết như người thật, không phải bot.

**Kết quả:** 100+ "genuine-looking" applications. Recruiting team nhìn vào không thấy nó generated by AI.

**Cái gì mình thích nhất:**

Santiago (dev xây cái này) không giữ nó riêng. Anh mở source code, share với cộng đồng. Tại sao? Vì anh hiểu: knowledge shared = society level up.

Đó là philosophy mình cũng believe. Mình share framework, mindset, template free. Khi bạn level up, business ecosystem level up, everyone wins.

**Bạn có thể learn từ Career-Ops:**

Nếu bạn xây bất kỳ business automation nào (outreach, customer service, data processing, content), structure nên là:

1. Collect data (scale)

## Post 5: The Technical Deep-Dive (For the Curious)

**Kỹ sư tâm lý:** Career-Ops work thế nào?

Career-Ops là một automation system built on Claude Code. Flow như sau:

**Pipeline:**
1. **Job Scraper** → Crawl 700+ listings từ job boards (LinkedIn, Indeed, etc.)
2. **Profile Analyzer** → Parse resume/skills của user, tạo skill vector
3. **Matcher** → Compare skill vector vs job requirements, output match score
4. **Resume Generator** → For high-match jobs, generate tailored resume (không copy-paste, generative AI tạo unique version)
5. **Cover Letter Generator** → AI viết cover letter reference job description + company culture
6. **Application Submitter** → Auto-fill forms + submit
7. **Tracker** → Monitor applications, responses, follow-ups

**Tại sao cách tiếp cận này effective:**

- **Scale:** 700+ listings mà human chỉ handle 10-20 application/week
- **Quality:** Tailored resume + cover letter vs generic template = 3-5x higher response rate
- **Speed:** 2-3 tuần execution vs 6 months manual search
- **Consistency:** System không mệt, không quên apply job tốt

**Santiago built cái này dùng:** Claude Code + API calls + job board integration

**Có gì mình respect:**

Anh ấy không bán tool này. Anh open source nó. Tại sao? Vì anh hiểu: technology giúp lợi ích cộng đồng mới là real success.

**Bottom line:**

Automation không phải magic. Nó là:
- Identify repetitive tasks
- Design system to handle them
- Execute at scale
- Iterate based on results
