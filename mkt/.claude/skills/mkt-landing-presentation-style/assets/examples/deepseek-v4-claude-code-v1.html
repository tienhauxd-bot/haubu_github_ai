05 · HOW IT WORKS

Bốn bước. Hết.

Không cần đổi tool, không cần học lại workflow. Vẫn Claude Code thân quen — chỉ thay base URL trỏ về proxy local.

01
Cài proxy

Clone free-claude-code và chạy 1 lệnh duy nhất để dựng local proxy ở port 8082.

$ git clone …/free-claude-code
$ cd free-claude-code
$ bun install
02
Set API key

Lấy DeepSeek API key, export vào env. Proxy sẽ tự convert sang Anthropic API format.

$ export DEEPSEEK_API_KEY="sk-…"
$ bun run start
# proxy listening :8082
03
Trỏ Claude Code

Mở terminal mới, set ANTHROPIC_BASE_URL trỏ về proxy rồi chạy Claude Code như bình thường.

$ export ANTHROPIC_BASE_URL="http://localhost:8082"
$ claude
04
Tiết kiệm 99%

Đó là tất cả. Workflow giữ nguyên, hoá đơn cuối tháng giảm từ $1,200 xuống ~$1.

06 · WHEN TO USE WHICH

Không thay thế.
Bổ sung.

Đây không phải trận đấu DeepSeek vs Claude — mà là biết khi nào nên dùng cái nào để tối ưu cả chi phí lẫn chất lượng.

// USE DEEPSEEK FOR
Bulk · Cheap · Fast
  • YESRefactor hàng loạt — đổi naming convention, thay API call cũ qua mới, cập nhật type signature.
  • YESScaffold project mới — sinh boilerplate, route, schema, CRUD pattern.
  • YESNarrow code task — viết unit test, regex, SQL query, parse JSON.
  • YESThrowaway script — file thử nghiệm, prototyping, automation 1 lần.
// USE CLAUDE FOR
Polish · Judgment · Long
  • YESArchitectural judgment — chọn tech stack, design pattern, trade-off scaling.
  • YESLong sessions — debug đa file, context lớn, suy luận xuyên suốt hàng giờ.
  • YESDesign taste & UX — UI polish, copywriting, brand voice, micro-interaction.
  • YESInstruction nuance — task có yêu cầu ngầm, ràng buộc tinh tế, an toàn cao.
// THE COMBO

Mở 2 terminal song song: dùng DeepSeek để cày 90% công việc, dùng Claude để review · polish · quyết định — chi phí giảm ~99% mà chất lượng vẫn ở mức Claude.
