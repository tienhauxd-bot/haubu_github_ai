05 · CÁCH DÙNG

Bốn bước. Hết.

Không cần đổi tool, không cần học lại workflow. Vẫn Claude Code thân quen — chỉ trỏ base URL về proxy local ở port 8082.

01
Tải về & cài

Tải free-claude-code từ GitHub, copy file env mẫu.

$ git clone github.com/Alishahryar1/free-claude-code
$ cd free-claude-code
$ cp .env.example .env
02
Đặt biến môi trường

Sửa file .env: chọn nhà cung cấp, dán API key, chốt MODEL.

NVIDIA_NIM_API_KEY="nvapi-…"
MODEL="nvidia_nim/z-ai/glm4.7"
ANTHROPIC_AUTH_TOKEN="freecc"
03
Chạy proxy

Khởi động FastAPI proxy ở port 8082 bằng uv.

$ uv run uvicorn server:app \
  --host 0.0.0.0 --port 8082
04
Trỏ Claude Code

Terminal mới — set 2 env var rồi chạy claude như bình thường.

ANTHROPIC_AUTH_TOKEN="freecc" \
ANTHROPIC_BASE_URL="http://localhost:8082" \
claude

Mẹo — mở 2 terminal: 1 chạy Claude thật cho việc khó, 1 chạy DeepSeek V4-Flash cho việc khối lượng lớn.
